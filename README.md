# CSpider
Spider with Web UI

### Main Third-party Libraries
* Python
  * Redis
  * Celery
  * Tornado
  * Sqlalchemy
* JS
  * Echart
  * JQuery
  * DataTables
* Database
  * Redis
  * MySQL

### Modules In CSpider
* Cecharts

  This Module Is Used To Pass The Rendered EChart Object To The Web. Maybe This Module Will Split From CSpider To Be Alone As A Library.
